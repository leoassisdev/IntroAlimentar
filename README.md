# IntroAlimentar

Aplicativo de acompanhamento de introdução alimentar para bebês com orientações da nutricionista pediátrica Gabrielly Botelho.

## Funcionalidades

- Cadastro e acompanhamento de bebês
- Registro de refeições com fotos
- Acompanhamento de alimentos alergênicos
- Relatórios semanais
- Sugestões de cardápio
- Guia de segurança alimentar
- Guia de emergência (engasgo, alergias)
- Calendário alimentar interativo

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS + Radix UI (shadcn/ui)
- Vite
- Electron (desktop app)

## Scripts

```bash
# Desenvolvimento web
npm run dev

# Build web
npm run build

# Desenvolvimento Electron (desktop)
npm run electron:dev

# Build .dmg (macOS)
npm run electron:build

# Build .exe (Windows)
npm run electron:build:win
```
